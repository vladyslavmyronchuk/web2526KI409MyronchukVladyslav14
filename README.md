LAB3 - WEBSOCKET

Hardware lab3:
Мікроконтролер ESP32 DevKit v1: Виступає як центральний обчислювальний модуль, що одночасно підтримує роботу HTTP-сервера та WebSocket-сервера.
Датчик магнітного поля HMC5883L: Цифровий трьохосьовий магнітометр, обраний відповідно до 4-го варіанту завдання.
Інтерфейс підключення I2C:GPIO 21 (SDA) — лінія даних.GPIO 22 (SCL) — лінія синхронізації
Модуль Wi-Fi: Забезпечує бездротовий зв'язок між клієнтом (браузером) та контролером для передачі HTTP-пакетів.

ПЗ та бібліотеки  lab3:
LittleFS : Енергонезалежна файлова система для автономного зберігання статичних ресурсів вебінтерфейсу  Flash-пам'яті.
WiFi.h : Забезпечує підключення до локальної мережі Wi-Fi для мережевої взаємодії.
WebSocketsServer v2.x: Реалізує серверний ендпоїнт на порті 81. Забезпечує повнодуплексний зв’язок, що дозволяє контролеру самостійно транслювати дані клієнту без постійних опитувань.
WebServer.h: Використовується для початкової видачі вебінтерфейсу на стандартному порті 80.ArduinoJson v7.2.2: Використовується для пакування значень осей X, Y, Z у формат JSON-пакета для передачі через WebSocket .

Frontend:
QMC5883LCompass v1.2.3: Спеціалізована бібліотека для ініціалізації датчика та зчитування цілочисельних значень магнітної індукції .
JavaScript (WebSockets API): Обробляє постійне з’єднання. Через обробник onmessage приймає JSON-пакети та оновлює стан інтерфейсу .
Chart.js (v4.x via CDN): Використовується для рендерингу лінійного графіка .
Алгоритм Rolling Window: Програмна реалізація зсуву графіка вліво шляхом видалення застарілих даних (понад 50 точок) для підтримки високої продуктивності системи .
